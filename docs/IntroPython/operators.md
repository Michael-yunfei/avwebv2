### 算数运算符

数学运算符，是为了对数进行运算，主要的算数运算符包括(主要适用于整数和小数):
<div class="code-example" markdown="1">

| 运算符     | 说明          | 案例 |
|:-------------|:------------------|:------|
| +           | 加法 | 2+3;  x + y  |
| - | 减法   | 7.2 - 5.9; x - y  |
| *           | 乘法     | 11 * 5.6; x * y   |
| /           | 除法 | 6/2; x/y  |
| %           |  取余  | 7%3; x%y |
|**           |  指数运算  |3\* \* 2; x\*\*y|
|//           |   整除  |9//4; x//y |

</div>

我们看一下具体的例子和代码:
```python
x = 6
y = 2

# 加、减、乘、除
print(x + y)  # 8

print(x - y)  # 4

print(x * y)  # 12

print(x / y)  # 3.0

# 取余
print(x % y)  # 0
print(9 % 2)  # 1

# 指数
print(x ** y)	# 36

# 整除
print(x // y)  # 3
print(9 // 2)  # 4
```

</div>
