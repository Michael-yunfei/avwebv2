### 简单的函数编写

函数的创建是为了可以重复性得调用模式化的程序体，从而更高效地解决不同问题。函数创建的三要素为:

- <span style="background-color: RGB(236, 66, 61)"> 输入和输出</span>
- <span style="background-color: RGB(236, 66, 61)"> 函数名</span>
- <span style="background-color: RGB(236, 66, 61)"> 函数体</span>

```python
def print_hello():
  print("Hello")
```
